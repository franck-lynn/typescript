# 如何定义泛型?

###  在函数中定义泛型

###### 声明式函数中定义

```ts
function fn<T>(arg: T): T{
    return arg
}
```

###### 箭头函数中定义

```ts
//1. 函数名 = <定义泛型>(参数: 定义的泛型): 返回值类型 => 函数体(返回值类型)
const fn = <T>(x: T): T => x
//2. 函数名: <泛型>(参数: 泛型) => 返回值类型 = (参数) => 函数体(返回值类型)
const fn: <T>(x: T) => T = x => x
//3. 函数名 = <泛型>(参数: 泛型): 返回值类型 => 函数体
const fn = <T>(x: T): T => x
// 第3种方式比较简单, 清晰
```
