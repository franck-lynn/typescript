#  tcp/ip协议

http 协议是 tcp/ip 协议之上, 即 nodejs 的 net 模块

由于都在 node 环境下运行, 不需要 tsconfig.json , 直接运行 模块即可

```json
 "scripts": {
    "dev": "nodemon -r esm client/src/index.ts",
    "server": "nodemon  -r esm server/src/main.ts"
  },
```
